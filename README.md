# 网页中外互译 - Chrome 扩展

> 中-英-外互转，双语显示。支持谷歌、微软等 20+ 翻译引擎。

## 功能特性

- 🌐 **20+ 翻译引擎**：微软、谷歌、百度、有道、DeepL、腾讯交互、阿里、Papago 等
- 📖 **双语对照显示**：原文 + 译文同时展示
- 🎨 **译文高亮**：自定义高亮颜色
- 🔍 **选词翻译**：鼠标选中文字即翻译
- ⚡ **输入框连击翻译**：输入框中连按三下空格翻译内容
- 📜 **滚动翻译**：页面滚动时自动翻译新内容
- 🔄 **自动翻译**：检测外文网页自动翻译
- 💾 **翻译缓存**：避免重复请求，提升速度
- 🖱️ **右键菜单**：右键选文翻译

## 安装方法

1. 下载或克隆本项目
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择 `SimpleTranslate` 文件夹

## 使用说明

### 基本操作

| 操作 | 说明 |
|------|------|
| 鼠标悬停侧边栏 | 展开翻译面板 |
| 点击 **转中文** | 将页面翻译为中文 |
| 点击 **转外文** | 将页面翻译为外语 |
| 点击 **双显** | 切换双语对照 / 单语显示 |
| 点击 **高亮** | 切换译文高亮 |
| 点击 **原文** | 切换显示原文/译文 |
| 点击 **选词** | 开启鼠标选词翻译 |
| 点击 **引擎** | 打开设置面板切换引擎 |

### 选词翻译

1. 点击侧边栏 **选词** 按钮开启
2. 用鼠标选中任意文字
3. 弹出翻译结果窗口

### 输入框翻译

在任意输入框中快速连按 **三次空格键**，自动翻译输入内容。

### 引擎切换

点击侧边栏 **引擎** 按钮打开设置面板，可：
- 从 20+ 引擎中选择
- 设置高亮颜色
- 开启/关闭滚动翻译
- 切换外语语种（部分引擎支持）

## 从油猴脚本迁移

本扩展由 [网页中英双显互译](https://greasyfork.org/zh-CN/scripts/469073) 油猴脚本重构而来，主要变化：

| 油猴脚本 | Chrome 扩展 |
|----------|------------|
| `GM_xmlhttpRequest` | Service Worker `fetch()` |
| `GM_setValue / GM_getValue` | `chrome.storage.local` |
| `GM_addStyle` | CSS 文件注入 |
| `GM_registerMenuCommand` | `chrome.contextMenus` |
| jQuery `$()` | 原生 DOM 操作 |

## 注意事项

- `icons/` 目录下的图标为占位文件，建议替换为正式图标（16×16、48×48、128×128 PNG）
- 部分翻译引擎（百度、有道、讯飞）需要自行申请 API 密钥并在代码中配置
- 谷歌翻译需要科学上网环境
- 排除的网站列表可在 `content.js` 中修改 `excludeSites` 数组
